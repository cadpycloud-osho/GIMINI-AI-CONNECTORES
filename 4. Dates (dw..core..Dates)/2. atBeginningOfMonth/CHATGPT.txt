atBeginningOfMonth:


example 1:


input:

@GetMapping("/atBeginningOfMonth1")
public Map<String, String> atBeginningOfMonth1() {
    OffsetDateTime dateTime = OffsetDateTime.parse("2020-10-06T18:23:20.351-03:00");

    OffsetDateTime result = dateTime
            .withDayOfMonth(1)
            .toLocalDate()
            .atStartOfDay()
            .atOffset(dateTime.getOffset());

    return Map.of("atBeginningOfMonthDateTime", result.toString());
}



output:

{
  "atBeginningOfMonthDateTime": "2020-10-01T00:00:00-03:00"
}
