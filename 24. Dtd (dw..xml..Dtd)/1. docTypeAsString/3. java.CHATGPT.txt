docTypeAsString:


example 1:


input:


public class Main {

    public static void main(String[] args) {
        String rootName = "cXML";
        String systemId = "http://xml.cxml.org/schemas/cXML/1.2.014/cXML.dtd";

        String docType = rootName + " SYSTEM " + systemId;
        System.out.println(docType);
    }
}


output:

cXML SYSTEM http://xml.cxml.org/schemas/cXML/1.2.014/cXML.dtd

example 2:



input:


public class Main {

    public static void main(String[] args) {

        String rootName = "html";
        String publicId = "-//W3C//DTD XHTML 1.0 Transitional//EN";
        String systemId = "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd";

        String docType = rootName + " PUBLIC " + publicId + " " + systemId;

        System.out.println(docType);
    }
}


output:


html PUBLIC -//W3C//DTD XHTML 1.0 Transitional//EN http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd
