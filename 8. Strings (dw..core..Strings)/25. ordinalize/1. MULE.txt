ordinalize:

ordinalize(num: Number): String

ordinalize(num: Number): String
Returns a number as an ordinal, such as 1st or 2nd.

%dw 2.0
import * from dw::core::Strings

{

  "c": ordinalize(5),
  "d": ordinalize(103)
}
