repeat:

repeat(text: String, times: Number): String

repeat(text: String, times: Number): String
Repeats a text the number of specified times.





Source
%dw 2.0
import * from dw::core::Strings
output application/json
---
{
  "a": repeat("e", 0),
  "b": repeat("e", 3),
  "c": repeat("e", -2)
}

DataWeave
Output
{
  "a": "",
  "b": "eee",
  "c": ""
}
