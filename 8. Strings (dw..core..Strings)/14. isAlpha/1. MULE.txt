isAlpha:

isAlpha(text: String): Boolean

isAlpha(text: String): Boolean
Checks if the text contains only Unicode digits. This excludes digits, punctuation, and other nonletter characters.

Note that the method does not allow for a leading sign, either positive or negative.

%dw 2.0

  "b": isAlpha(""),
  "c": isAlpha("  "),

  "e": isAlpha("ab2c"),
  "f": isAlpha("ab-c")
}
