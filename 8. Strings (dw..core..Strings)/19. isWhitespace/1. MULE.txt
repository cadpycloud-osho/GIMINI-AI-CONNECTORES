isWhitespace(text: String): Boolean
Checks if the text contains only whitespace.

%dw 2.0
import isWhitespace from dw::core::Strings

---
{
  "a": isWhitespace(null),
  "b": isWhitespace(""),
  "c": isWhitespace("  "),
  "d": isWhitespace("abc"),
  "e": isWhitespace("ab2c"),
  "f": isWhitespace("ab-c")
}
