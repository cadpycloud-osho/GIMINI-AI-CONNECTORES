        String[] users = {"Mariano", "Leandro", "Julian"};
        String target = "Julian";
        int index = -1;
        
        for (int i = 0; i < users.length; i++) {

Using Java's List.indexOf method, which matches DataWeave behavior exactly.

import java.util.Arrays;
import java.util.List;

public class IndexOfList {
    public static void main(String[] args) {
        List<String> users = Arrays.asList("Mariano", "Leandro", "Julian");
        int index = users.indexOf("Julian");
        System.out.println(index);
    }
}
Output: 2
