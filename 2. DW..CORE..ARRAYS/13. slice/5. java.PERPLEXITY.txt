Java Array Copy Version
Uses Arrays.copyOfRange which matches DataWeave slice behavior exactly (start inclusive, end exclusive).

java
import java.util.Arrays;

public class SliceArrayCopy {
    public static void main(String[] args) {
        int[] arr = {0, 1, 2, 3, 4, 5};
        int[] result = Arrays.copyOfRange(arr, 1, 4);
        System.out.println(Arrays.toString(result));
    }
}
Output: [1, 2, 3]

Java Streams Version
Functional slicing using Streams with skip and limit operations.

java
import java.util.Arrays;

public class SliceStream {
    public static void main(String[] args) {
        int[] arr = {0, 1, 2, 3, 4, 5};
        int[] result = Arrays.stream(arr)
                            .skip(1)
                            .limit(3)
                            .toArray();
        System.out.println(Arrays.toString(result));
    }
}
Output: [1, 2, 3]
