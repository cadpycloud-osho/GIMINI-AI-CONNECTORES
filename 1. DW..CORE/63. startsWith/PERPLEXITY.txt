1. Check if "Mari" starts with "Mar"
java
@RestController
class Example1_StartsWithMar {

    @GetMapping("/startsWithMar")
    public boolean startsWithMar() {
        String input = "Mari";
        return input.startsWith("Mar");
    }
}

true

Bonus: Combined example returning both results as a list

@RestController
class Example3_StartsWithCombined {

    @GetMapping("/startsWithCombined")
    public List<Boolean> startsWithCombined() {
        String input = "Mari";
        return List.of(input.startsWith("Mar"), input.startsWith("Em"));
    }
}
Output:

json
[true, false]
